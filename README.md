# 👟 ALDO STORE PRO

ALDO STORE PRO is a simple online shop web app built using HTML, CSS, and JavaScript.  
It includes an Admin Panel to manage products easily.

---

## 🚀 Features

- 👕 Add products from Admin Panel  
- 🗑️ Delete products instantly  
- 📂 Categories system:
  - Shoes 👟  
  - Clothes 👕  
  - Accessories 💍  
- 💾 Save products using LocalStorage  
- 📱 WhatsApp order system  
- 🔐 Admin panel with password protection  
- ⚡ Instant updates without refresh  

---

## ⚙️ How to Use

1. Open the website  
2. Click on **Admin button ⚙️**  
3. Enter admin password  
4. Add product (name, price, category, image URL)  
5. Products will appear instantly  

---

## 🛠️ Tech Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  

---

## ⚠️ Important Notes

- Products are saved in browser (LocalStorage)  
- If browser data is cleared, products will be lost  
- No backend or database is used
